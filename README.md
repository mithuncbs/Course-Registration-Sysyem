Modern universities handle hundreds of students across multiple departments, each of 
whom must register for courses every semester. Managing this process efficiently is critical 
to ensuring students are enrolled correctly, sections are not over-filled, and academic 
records remain accessible and accurate throughout a student's academic career. 
Manual or spreadsheet-based registration systems are prone to errors such as duplicate 
records, lost data, and the inability to quickly query a student's academic history. As the 
number of students grows, such methods become increasingly impractical. A software 
solution built on the right data structures can eliminate these issues while being both 
scalable and easy to extend. 
This project presents the design and implementation of a University Course Registration 
System developed in the C programming language. The system uses a multi-level Linked 
List structure to manage student records, semester registrations, and course enrolments 
dynamically. The project was developed as part of the Data Structures Lab course (CSE 
221) at Daffodil International University.
